# debox-chat-go-sdk

This is a Golang SDK for Debox Chat Service.

API Reference :

* [English](https://help.debox.pro/openapi_cn/o/summary)

# Install Instruction

### Debox Chat Golang SDK

```
go get -u github.com/debox-pro/debox-chat-go-sdk
```
